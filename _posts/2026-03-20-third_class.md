## 📚 학습 내용
세번째 수업을 했다. 오늘은 금요일이라 조금 일찍 끝내주신것 같다. 

여전히 select 문을 배우고 있다. 
문자열 출력에 관해 배우면서는 instr, substr, trim, ltrim, rtrim, replace 등의 함수가 나왔는데 예전에 많이 써먹던 split 함수가 떠올랐다. 
이렇게 문자열을 다루는 함수들이 많다니 갈길이 멀구나 싶었다.

이후로는 숫자 함수에 대해 배웠는데 기억이 날듯 말듯 하면서도 헷갈리는 부분들이 있었다.
다른건 제처두더라도 mod, power, abs, sqrt 이것정도는 다시 확실히 외워두자.

마지막으로 날짜 함수에 대해 배웠다. 
단순히 외워햐 할 정도이고 써야할게 많아서 햇갈리긴 한데 복잡하진 않았다.
이정도로 모르고 있었다니 반성하자.

수업 마지막에 낸 문제를 풀어보았는데 
문제는 짝수달에 입사한 사원들의 정보를 출력하는 것이였다.

사실 mod 함수를 사용하면 쉽게 풀 수 있는 문제였는데 
까먹고 round, trunc 함수를 사용해서 풀었다. 

내 풀이는 다음과 같다.

```sql
select *
from hr.employees
where round(to_char(hire_date, 'mm')/2) = trunc(to_char(hire_date, 'mm')/2);
```
이렇게 비효율의 극치를 보여주었다.
좀 많이 민망스럽다. 제대로 알지 못하여 아는것만 사용해서 풀려고 하니 이렇게 되었다.

강사님의 정답은 다음과 같다.

```sql
select *
from hr.employees
where mod(to_char(hire_date, 'mm'), 2) = 0;
```
얼마나 깔끔한가. 복습하자.


## 🎯 오늘 배운 핵심 내용
- mod 함수는 나머지를 반환한다.
- to_char 함수는 date 타입을 문자열로 변환한다.
- 숫자형과 문자형은 암시적으로 형변환이 가능하여 비교가 가능할때가 있다.
